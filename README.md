# Python Generators - Database Seeding

This script sets up a MySQL database `ALX_prodev` with a `user_data` table and seeds it from `user_data.csv`.

## Requirements
- MySQL Server running on localhost
- Python 3
- `mysql-connector-python`: `pip install mysql-connector-python`

## Files
- `seed.py`: Contains all database setup and seeding logic
- `user_data.csv`: Sample data
- `0-main.py`: Test script

## Usage
```bash
./0-main.py
